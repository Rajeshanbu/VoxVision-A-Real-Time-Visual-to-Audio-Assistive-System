# VoxVision: A Real-Time Visual-to-Audio Assistive System

A web-based assistive application that helps visually impaired users navigate, understand surroundings, and read text using real-time computer vision and voice feedback.

---

## About

VoxVision AI is an accessibility-focused project that uses Artificial Intelligence, Computer Vision, and Web Speech technologies to convert real-world visual information into spoken guidance. Traditional assistive devices are often expensive and inaccessible. This project provides an affordable alternative that runs directly in a web browser using a smartphone or laptop camera.

The system captures live video, detects obstacles, describes surroundings using AI, reads printed text using OCR, and provides hands-free voice guidance. VisionAID AI aims to empower visually impaired individuals with independent navigation and environmental awareness.

---

## Features

- Real-time obstacle detection using TensorFlow.js  
- AI scene description using Hugging Face Vision API  
- Text recognition using Tesseract OCR  
- Voice feedback using Web Speech API  
- Voice commands for hands-free control  
- Runs directly in browser (No installation required)  
- Works on mobile and laptop devices  

---

## Requirements

* **Operating System:** Windows 10/11, Linux, or Android device  
* **Development Environment:** Node.js (v16 or later)  
* **Frontend Framework:** React.js with Vite  
* **AI Libraries:** TensorFlow.js, COCO-SSD  
* **OCR Library:** Tesseract.js  
* **Cloud AI:** Hugging Face Inference API  
* **Browser APIs:** Web Camera API, Web Speech API  
* **Version Control:** Git & GitHub  
* **IDE:** VS Code recommended

## Results and Impact

VisionAID AI successfully demonstrates real-time navigation, scene understanding, and text reading using web-based AI technologies. The system enables hands-free assistance, improves environmental awareness, and provides an affordable accessibility solution without specialized hardware.

This project contributes to inclusive technology development and highlights the potential of browser-based AI for real-world assistive applications.
